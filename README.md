# 🔐 ChronoVault – Secure Cloud-Based Self-Destructing File Vault

A secure full-stack file vault built with React and Flask that uses AES-GCM encryption, one-time access keys, audit logging, and automatic file expiry to protect sensitive data.

## 📌 Overview

ChronoVault is a secure cloud-based file vault designed to protect sensitive files through encryption, controlled access, and automated data lifecycle management.

The system encrypts files before storage, generates secure access keys, maintains audit logs, and automatically removes files after their expiry period.

---

## 🚀 Key Features

- 🔐 AES-GCM encryption for secure file storage
- ⏳ Automatic file expiry and self-destruction
- 🔑 One-time secure access key generation
- 📜 Audit logging for file activities
- ☁️ Cloud-ready storage architecture
- 📤 Secure file upload and retrieval
- 🧩 Full-stack implementation using React and Flask
